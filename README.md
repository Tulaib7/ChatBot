AI SQL Chatbot (PostgreSQL Powered):
A Python based AI chatbot that connects to a PostgreSQL database and lets users fetch data using natural language.
Non technical users can simply ask questions, and the bot automatically generates the correct SQL query, executes it, and returns results in a clean table format.

Features:
-Understands natural language queries
-Converts prompts into valid PostgreSQL SQL queries
-Executes queries securely
-Fetches data from any table in the database
-Returns results in a clean formatted table
-Helpful for data teams who want quick access without running SQL manually

Tech Stack:
-Python
-Bytez GPT model
-PostgreSQL
-psycopg2
